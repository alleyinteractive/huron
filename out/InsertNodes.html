Class: InsertNodes

InsertNodes

Class for inserting HTML snippets at particular insertion points. Uses require() to grab html partials, then inserts that html into an element with attribute [huron-id] corresponding to the reference URI of the target KSS section, and [huron-type] corresponding with the required KSS field
